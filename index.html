HTML

HTML, czyli HyperText Markup Language, jest standardowym językiem znaczników wykorzystywanym do tworzenia stron internetowych. Jest to język oparty na tagach, które definiują strukturę i zawartość dokumentu internetowego. HTML jest fundamentalnym narzędziem w tworzeniu stron internetowych, umożliwiającym organizację treści, tworzenie hiperłączy, dodawanie obrazów, formularzy i innych elementów interaktywnych.

CSS

CSS, czyli Cascading Style Sheets, jest językiem stylów używanym do opisu wyglądu i formatowania stron internetowych oraz dokumentów HTML. Jest to kluczowa technologia w tworzeniu atrakcyjnych i responsywnych stron internetowych, która umożliwia projektowanie interfejsów użytkownika, dostosowywanie stylów do różnych urządzeń oraz tworzenie efektów wizualnych.

JavaScript & jQuery

JavaScript to język programowania powszechnie stosowany do tworzenia interaktywnych stron internetowych. Jest on wykorzystywany zarówno po stronie klienta (w przeglądarkach internetowych), jak i po stronie serwera (za pomocą platform takich jak Node.js). JavaScript umożliwia dodawanie interakcji, dynamiczne modyfikowanie zawartości strony, obsługę zdarzeń użytkownika oraz komunikację z serwerem.
jQuery to biblioteka JavaScript, która ułatwia pracę z JavaScriptem poprzez zapewnienie zestawu prostych i wygodnych funkcji do manipulacji DOM, obsługi zdarzeń, animacji oraz komunikacji z serwerem. Główne zalety jQuery to uproszczenie składni, kompatybilność oraz wszechstronność.

Bing Create

Obrazy zostały wygenerowane przez Bing Create, korzystającego z modelu DALLE-3 "DALLE-3" to zaawansowany model sztucznej inteligencji, który został wykorzystany do generowania obrazów przez narzędzie Bing Create. Model ten jest częścią rodziny modeli DALL-E (ang. "DALL-E" - "Data Augmented Language Learning with Empathy"), które zostały opracowane przez OpenAI. DALL-E wykorzystuje architekturę sieci neuronowej do generowania obrazów na podstawie opisów tekstowych, co umożliwia tworzenie realistycznych i kreatywnych grafik na podstawie abstrakcyjnych koncepcji. Dzięki zaawansowanym zdolnościom generatywnym, model DALLE-3 może tworzyć złożone, wysokiej jakości obrazy, które mogą być wykorzystane w różnych kontekstach, od sztuki po projektowanie interfejsów użytkownika.
